# Energy Management System (EMS)

An extensible home energy management system.

## Features

- Simple and clean user interface
- Support for many energy meters and solar inverters
- Support for many smart switches and heating devices
- Local energy management without relying on cloud services
- Modbus, HTTP, MQTT, WebSocket, and script integrations
